# AI Fraud Detection in Cryptocurrency using Machine Learning

## Project Overview

This project uses Machine Learning algorithms to detect fraudulent cryptocurrency transactions.

The system compares multiple machine learning models to identify suspicious transaction patterns.

## Algorithms Used

- Logistic Regression
- Decision Tree
- Random Forest
- XGBoost
- Support Vector Machine

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- Matplotlib

## Evaluation Metrics

- Accuracy
- Precision
- Recall
- F1 Score
- AUC ROC

### Install Libraries
